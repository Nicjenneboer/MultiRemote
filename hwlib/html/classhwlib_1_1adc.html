Detailed Description

A/D input interface

This class abstracts the interface to an ADC (Analog to Digital Converter).

◆ read()

virtual adc_value_type hwlib::adc::read ( )
pure virtual

do an A/D conversion and return the result

This function performs an A/D conversion and returns the result. The lower n_bits of the result are the conversion result, the remaining higher bits are 0.

Implemented in due::pin_adc, db103::pin_adc, and atmega328::pin_adc.

◆ refresh()

virtual void hwlib::adc::refresh ( )
pure virtual

refresh the adc value

This function refreshes the contents of the read buffer that might be used by a buffered read().

That buffer might also be refreshed as a side-effect of other operations.

Implemented in due::pin_adc, db103::pin_adc, and atmega328::pin_adc.
