virtual uint_fast16_t hwlib::port_in::read ( )
pure virtual

read from the port

This function reads and returns the pins that are part of the port. The lowest bit of the result reflects the first pin of the port, etc.

Before calling this function the port direction must have been set to input by calling port_direction_set_input().

virtual void hwlib::port_in::refresh ( )

refresh the port buffer

This function refreshed the contents of the read buffer that might be used by get_buffered() with a value read from the pins.

The read buffer might also be refreshed as a side-effect of other operations.
