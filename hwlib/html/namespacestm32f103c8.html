Detailed Description

hwlib HAL for the stm32f103c8

This namespace contains the hwlib implementation of the pins, timing and (software) UART output for the stm32f103c8.

Initially, the chip runs from its HSI internal RC oscillator at 8 MHz. The first wait call configures the chip to run at 72 MHz, assuming an 8 MHz crystal.

The chip runs at 3.3 Volt and that is the level on its IO pins.

  • STM32Fl03x8 datasheet (pdf)
