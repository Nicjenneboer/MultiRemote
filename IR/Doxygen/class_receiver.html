Detailed Description

Receiver.

This Receiver class is a child class from IR_Data.
The Receiver get signals from the pin_in.
These signals can be decode() with the associated protocol.

Member Function Documentation

◆ decode()

void Receiver::decode ( )

Decode the raw_data with the associated protocol.

This decode() function compares sequently after the start signals if two digitial signals are inRange() with the logic one signal in the protocol.
If it's true, a one will be set in the decode code, else an zero.

◆ detect()

void Receiver::detect ( unsigned int  time = 0)

Detect an IR signal.

This detect() function waits until the pin_in makes a digital signal switch.
The time between two signal switches and the signal state would be stored in an array as raw_data.

◆ match()

void Receiver::match ( )

Find the associated protocol from stored raw_data.

This match() function compares raw_data:
1) The amount bits from the protocol.
2) If the logic start signal is inRange() with the protocol logic start signal.
3) If all the digital signal state time is inRange() with the protcol.
With all protocols.
