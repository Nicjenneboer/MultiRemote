Detailed Description

Transmitter.

This Transmitter class is a child class from IR_Data.
Stored code with a matching protocol can be encoded to digital signals.
These signals can be send to the pin_out.

Member Function Documentation

◆ encode()

void Transmitter::encode ( )

Encode the stored code with the associated protocol.

This encode() function sets the first two signals from encoded_data with the protocol::start signals.
Then for all bits in code:
if bit is one put at the end off encoded_data two signals from protocol logic one
if bit is zero put at the end off encoded_data two signals from protocol logic zero.

◆ send()

void Transmitter::send ( )

Send stored encoded_data with 38kHz to pin_out.

This send() function send for all encoded_data signals:
If encoded_data signal state is one: a high and low digital signal on pin_out for 13 microseconds(38kHz) for x encoded_data signal time.
If encoded_data signal state is zero: a low digital signal on pin_out for x encoded_data signal time.
